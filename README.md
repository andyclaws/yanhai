
## 工程结构
``` 
sea-web-admin
├── public -- 第三方资源
├── src -- 项目代码
├    ├── api -- api接口配置
├    ├── assets -- 静态资源
├    ├── components -- 自定义公共组件
├    ├── config -- 配置
├    ├── directive -- 自定义指令
├    ├── libs -- 工具类
├    ├── locale -- 语言包
├    ├── mock -- mock静态数据
├    ├── plugin -- 插件
├    ├── router -- 路由
├    ├── store -- 状态管理
├    └── view -- 页面
└──  tests --
```
